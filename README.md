# Projects – Informatica Medica

Raccolta dei task svolti per il corso di Informatica Medica, in gruppo (Gruppo 13). Ogni cartella contiene script, output e un proprio README con la spiegazione dettagliata.

## Struttura

### [`task1/`](./task1) — Rimborsi ospedalieri per DRG
Calcolo dell'introito ospedaliero a partire dai DRG dei ricoveri (con tariffe differenziate per durata di degenza) e di indicatori di controllo qualità (degenze anomale, trasferimenti).

### [`task2/`](./task2) — Cifratura RSA
Implementazione di una cifratura a chiave pubblica di tipo RSA su un messaggio testuale, con codifica a blocchi ed esponenziazione modulare.

### [`task3/`](./task3) — Compressione di immagine tipo JPEG
Compressione a blocchi 8×8 basata su DCT (Discrete Cosine Transform) e quantizzazione, applicata a un'immagine ecografica, con codifica differenziale della componente DC e troncamento della componente AC.

### [`task4/`](./task4) — Query SQL e progettazione database
Interrogazioni SQL su uno schema sanitario esistente e progettazione di un nuovo schema (tabelle, chiavi, vista) per la gestione di immagini diagnostiche.

## Requisiti

MATLAB (task 1–3), un DBMS SQL standard (task 4).
